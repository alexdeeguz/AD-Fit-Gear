# ADFitGear
[View the live site](https://ad-fit-gear.herokuapp.com/#/)

ADFitGear, inspired by BMFitGear, is an e-commerce store where users can view and buy product listings for fitness gear and apparel. 

## Technologies Used
### Backend
* Ruby on Rails
* PostgreSQL
### Frontend
* React/Redux
### Other
* Rails Active Storage
* Amazon S3
* SCSS
* Webpack
* Heroku
